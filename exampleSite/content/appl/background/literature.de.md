Die Offenbarung ist zugleich Brief, Prophetie und Apokalypse — das heißt, sie wurde geschrieben, um als echte Botschaft an echte Menschen gelesen zu werden, gemeint, jetzt eine Veränderung hervorzurufen, nicht als Rätsel über weit entfernte Ereignisse. Prophetie zielte nie einfach darauf ab, die Zukunft vorherzusagen; sie zielte darauf ab, die Gegenwart zu verändern, damit die angekündigte Zukunft nicht eintreten musste — wie bei Jonas Warnung an Ninive.

Das ist eine ganz andere Art, dieses Buch zu halten, als es wie einen zu knackenden Code zu behandeln. Ein Code muss nur gelöst werden. Ein Brief will befolgt werden.

- Hast du die Offenbarung (oder Prophetie im Allgemeinen) bisher eher als Rätsel gelesen, das es zu lösen gilt, oder als einen Brief, der persönlich an dich gerichtet ist und gerade jetzt etwas von dir verlangt?
- Wenn eine bestimmte Stelle in der Offenbarung dazu gedacht ist, heute eine Veränderung in dir hervorzurufen, welche Stelle fällt dir zuerst ein — und welche Veränderung fordert sie tatsächlich?
